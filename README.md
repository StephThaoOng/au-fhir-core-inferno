# AU Core Test Kit

## Dev version
Please try dev version here http://au-inferno.beda.software/

## Generate tests

1. Put your IG in the folder
```bash
/lib/au_core_test_kit/igs
```
2. Execute ruby script
```bash
make generate
```

## Run inferno
1. Run setup script
```bash
make setup
```
2. Run
```bash
make run
```

## Run production
```bash
make run-production
```
