# AU Core Test Kit

## Dev version
Please try dev version here http://au-inferno.beda.software/

## Generate tests

1. Put your IG in the folder
```bash
/lib/au_core_test_kit/igs
```
2. Execute ruby script
```bash
bundle exec rake au_core:generate
```

## Run inferno
1. Run setup script
```bash
./setup.sh
```
2. Run
```bash
./run.sh
```
